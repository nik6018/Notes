
Divide and Conquer Technique 
----------------------------
	
	(Binary search)

		1) Pick a Midpoint 

		2) Check if the value is equal to the given value

		3) Otherwise remove the unwanted part of the array and repeat the above steps till the value is found

	(Merge sort)

		1) Divide the problem into half size 

		2) And keep dividing array until the base case 

		3) Start merging until the orignal case is achieved


Loop Invariant (Selection sort , Bubble sort)
--------------

	1) Pick an element from	 the array

	2) Compare it to the all the other element to its right in the array

	3) Move the element to the proper position and repeat
